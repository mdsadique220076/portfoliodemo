Website for a restaurant

I created this project from a tutorial and little by myself.

The objectve : Gain more visibilty for the restaurant and display the pizza menu to the users.

This project is simple and efficient. In this project the website owner does not have to edit or add content by editing html file, the admin can easily add products in pizza menu by going through the admin page of the website.

Future objective : Give more ease to the admin, so that the admin can also add differnt menus like Pasta Menu, Chicken Menu or else without going through html file.

Technologies : HTML, CSS, DJANGO, PYTHON, HEROKU
